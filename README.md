# FIAP Bank ATM 

Projeto desenvolvido para a disciplina de Domain Driven Design (Java), com o objetivo de simular um terminal de autoatendimento bancário (ATM).

## Estrutura do Projeto

O código-fonte principal está localizado no seguinte diretório:
src/main/java/org/example

Dentro dessa pasta encontra-se a classe principal:

- `FiapBankAtm.java` → responsável pela execução do sistema
